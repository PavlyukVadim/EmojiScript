# EmojiScript

The basic concept of emoji based programming language that transpires into javascript :)

## Code samples:

##### EmojiScript

```js
i set 10
🔁(i > 0) {
  i--
  🤔(i is 👍) {
    📝("Some text")
  }
}
```

##### JavaScript

```js
let i = 10
while (i > 0) {
  i--
  if(i == true) {
    console.log("Text")
```
